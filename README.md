# Ball_Catch_Game (Python Tkinter + Voice)

A fun and interactive desktop game built using **Python's Tkinter GUI** and **pyttsx3** for real-time **voice feedback**! Move your basket to catch falling eggs ‚Äî and hear your score and remaining lives spoken aloud!

## Gameplay Features

-  Control the catcher using `‚Üê` and `‚Üí` arrow keys.
-  Voice announcements using `pyttsx3` (score and lives).
-  Miss an egg? Lose a life.
-  Difficulty increases over time (faster Balls).
